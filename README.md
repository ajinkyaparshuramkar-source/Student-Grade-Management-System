# Project Statement

Problem Statement:
Teachers struggle when they have to keep track of student marks by hand. It is hard for them to calculate grades or find the best students without a computer. Doing this work manually takes a long time and it is easy to make mistakes

Scope of the Project:
This project aims to build a simple, console-based "Student Grade Management System" that allows users to digitally record student names and marks. It stores data persistently and performs basic analytics.

Target Users:
* Teachers
* Lab Instructors
* School Administrators

High-Level Features:
1.  Data Entry: marks and students name can be added securly.
2.  validation: marks should be in the range 0-100 and only numeric values.
3.  Persistence: if we exit the data will store 
4.  Analytics: Just after that calculate class average and find the class topper.
